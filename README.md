# Dwell at will

This project is a multimedia artwork that blends sound, image, and conceptual design.  
It explores the intersection of technology and creativity, presenting an experimental approach to artistic expression.  

## Features
- Audio-visual compositions inspired by literature and philosophy  
- Digital sketches and edited images  
- Soundscapes created with professional tools  

## Usage
Feel free to explore the files, listen to the soundscapes, and view the images.  
The project is intended for educational, artistic, and portfolio purposes.
